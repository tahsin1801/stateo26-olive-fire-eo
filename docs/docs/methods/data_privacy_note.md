# Data and Privacy Note

This repository provides non-confidential supplementary material for StatEO26 Poster #35 / Abstract 144.

The case study was developed in a court-oriented and public-authority context. For this reason, the repository intentionally does not include raw spatial evidence or restricted case-file material.

## Materials not shared publicly

The following materials are not included in this public repository:

- Raw UAV imagery
- Orthophoto, DSM, DTM, CHM, or other full-resolution GeoTIFF products
- LAZ, point-cloud, 3D model, or textured mesh files
- Cadastral parcel polygons or TKGM-derived coordinate datasets
- Full coordinate lists of individual tree locations
- Judicial records, expert reports, petitions, or case-file documents
- Personal data or party-identifying information

## Materials shared publicly

The repository may include:

- Methodological notes
- Workflow summaries
- Public-safe code outlines
- Generalized processing logic
- Non-confidential summary statistics
- Low-risk figures prepared for scientific communication

## Reproducibility approach

The repository is designed to document the methodological structure of the workflow without exposing restricted spatial, cadastral, or judicial evidence.

The aim is to make the analytical reasoning understandable and reviewable while preserving confidentiality, data protection, and case-file integrity.

## Disclaimer

This repository does not constitute a public release of the underlying case evidence.

All shared material is intended for scientific communication and methodological transparency only.
